
# UWSN Federated Learning System

## Problem Description
Controlled causal UWSN simulation using:
- Federated Learning
- Digital Twin
- Continual Learning
- Adaptation

## Setup
pip install -r requirements.txt

## Run
python project/src/main.py

## Experiments
exp1_baseline.py -> Full System
exp2_no_adapt.py -> No Adaptation
exp3_no_fl.py -> No Federated Learning
exp4_drift.py -> Drift Scenario

## Outputs
- CSV metrics
- Statistical validation
- Ablation study
- Error bar plots
- Correlation analysis
